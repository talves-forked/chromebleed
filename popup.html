This extension checks to see if the site you are currently using is affected by the Heartbleed bug.

Special thanks go to our Other Contributors also.
It is based on the excellent work of Filippo Valsorda.

This tool should have a limited lifespan as the sites you use patch their servers. The developers needed some redundancy in deploying. Stopbleed with follow the exact features of the original versions. This release is for redundancy and exposure.
