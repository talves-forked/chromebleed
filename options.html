Stopbleed

The Chromebleed extension checks to see if the site you are currently using is affected by the Heartbleed bug.

Official release from Tony Alves.
This tool was originally developed by Jamie Hoyle
Special thanks go to our Other Contributors also.
It is based on the excellent work of Filippo Valsorda.

You can contribute or get the latest bleeding edge changes on Github

Options



This tool should have a limited lifespan as the sites you use patch their servers. The developers needed some redundancy in deploying. Stopbleed with follow the exact features, functionality and version numbers of the original versions. This release is for redundancy and exposure.
